\# Frontend Design Playbook

\- Maintain strict, structural spacing scales using modern framework tokens (like Tailwind rems).

\- Prioritize striking typography hierarchies over generic browser defaults.

\- Enforce clean, intentional color choices that comfortably meet WCAG accessibility standards.
